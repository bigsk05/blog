原文概要：
Aircraft registration number recognition can be used in the fields of photo classification and rapid detection of aircraft flight information, providing convenient automation solutions for these fields. This study synthesizes the problems of object recognition, character recognition, sorting and retrieval of messy data, and puts forward reasonable solutions for these problems. Finally, a computer vision system based on deep learning and global aircraft database retrieval were realized, which can automatically identify the aircraft in the image and return detailed information.
